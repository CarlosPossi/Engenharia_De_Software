# 📚 Portfólio — Engenharia de Software | FIAP 2026

## Sobre este repositório

Este repositório reúne os exercícios, diagramas, documentos e atividades desenvolvidos durante a disciplina de Engenharia de Software | FIAP em 2026.

O objetivo deste portfólio é documentar a evolução prática dos conteúdos estudados em sala de aula, incluindo levantamento de requisitos, modelagem UML, arquitetura de software e documentação técnica.

Além disso, o repositório serve como apoio para revisão dos conteúdos e demonstração prática das habilidades desenvolvidas ao longo da disciplina.

---

## ▶️ Como executar os exercícios

### Pré-requisitos

Antes de começar, você precisará ter instalado em sua máquina:

* Git
* Navegador Web
* Editor de código (recomendado: VS Code ou Google Colab)
* Java JDK (quando necessário)
* Draw.io ou StarUML para visualizar diagramas UML

### Instalação

Clone o repositório:

```bash
git clone https://github.com/CarlosPossi/Engenharia_De_Software.git
```

Acesse a pasta do projeto:

```bash
cd Engenharia_De_Software
```

Abra os arquivos no VS Code ou em outro editor de sua preferência.
